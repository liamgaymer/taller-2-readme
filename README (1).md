# Sistema Academico - Gestion de Estudiantes y Profesores

Este es mi proyecto de python para gestionar estudiantes y profesores desde la consola.

## ¿Que hace el programa?

- Puedes agregar, ver y eliminar estudiantes
- Puedes agregar, ver y eliminar profesores
- Muestra estadisticas generales (cuantos hay, promedio de edad, salarios, etc)
- Guarda todo en archivos .json para que no se pierda la informacion

## Como correrlo

1. Clona o descarga el proyecto
2. Entra a la carpeta del proyecto
3. Corre el archivo principal:

```
python main.py
```

## Estructura de carpetas

```
proyecto/
```

- Python 3.8 o superior
